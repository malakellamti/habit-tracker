# Habit Tracker 

A web application to track daily habits with streaks and statistics.

## Features
-  Add and manage daily habits
-  Streak tracking
-  Weekly/monthly statistics with heatmap
-  Dark mode
-  Responsive design

## Tech Stack
- Vue 3 (Composition API)
- Pinia
- Chart.js
- localStorage

## Getting Started

### Install dependencies
npm install

### Run the app
npm run dev
